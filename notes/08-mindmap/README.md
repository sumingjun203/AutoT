## 🔍 查漏补缺清单

### 阶段一（Week 2 结束）
- [ ] 能否不看文档写出一个 pytest 项目？
- [ ] 能否解释 fixture scope 的区别？
- [ ] 能否用 parametrize 实现数据驱动？

### 阶段二（Week 6 结束）
- [ ] 能否独立设计 POM 三层架构？
- [ ] 能否处理 iframe、alert 等特殊场景？
- [ ] 能否用 YAML 做数据驱动？

### 阶段三（Week 10 结束）
- [ ] 能否独立搭建完整 API 测试框架？
- [ ] 能否处理 Token 鉴权 + 接口关联？
- [ ] 能否用 mock 解耦第三方依赖？

### 阶段四（Week 12 结束）
- [ ] 能否配置 GitHub Actions 自动跑测？
- [ ] 能否生成 Allure 报告并部署？
- [ ] 能否用日志 + 截图定位失败原因？

### 阶段五（Week 13 结束）
- [ ] 2 个 GitHub 仓库 README 完整
- [ ] 50 面试题能流畅回答
- [ ] 简历更新 + 项目经验写好
