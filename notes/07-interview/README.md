## 🎯 面试准备策略

### 简历项目
每个项目准备 5 分钟讲解：
1. **项目背景** — 解决什么问题
2. **技术选型** — 为什么选 pytest + Selenium
3. **架构图** — 分层怎么设计
4. **量化成果** — 多少用例、节省多少时间
5. **难点突破** — 遇到什么问题、怎么解决的

### 常见面试问题
- 你如何设计一个自动化测试框架？
- pytest fixture 的 scope 有哪些？区别？
- Selenium 元素定位失败怎么排查？
- 接口依赖怎么解耦？
- 测试报告怎么选型？
- 自动化测试覆盖率如何提升？
- 如何说服团队采用自动化测试？

### 准备清单
- [ ] 简历项目经验写好（项目 + 技术 + 量化）
- [ ] 2 个 GitHub 公开项目有完整 README
